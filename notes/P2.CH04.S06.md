# Cauchy-Schwartz inequality

$$
|\mathbf{u} \cdot \mathbf{v}| \leq |\mathbf{u}| |\mathbf{v}|
$$

# Triangle inequality

$$
|\mathbf{u} + \mathbf{v}| \leq |\mathbf{u}| +  |\mathbf{v}|
$$

### Note that all we have done generalizes to $\mathbb{R}^n$
